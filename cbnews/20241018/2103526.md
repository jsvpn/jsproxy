href="https://www.bannedbook.org/bnews/tag/%e5%93%88%e4%bd%9b%e5%a4%a7%e5%ad%a6/" class="st_tag internal_tag" rel="tag" title="标签 哈佛大学 下的日志">哈佛大学</a>校园再度成为政治抗议与争议的焦点。根据《华盛顿自由灯塔》( Washington Free Beacon ) 报导，4 月 20 日，在哈佛肯尼迪学院举办的一场活动中，来自台湾的学生吴可薇 ( Cosette Wu ) 打断了中国驻美国大使谢锋的演讲，指责中国剥夺香港人民的基本自由，并试图对台湾进行类似的行动。这场抗议短暂但有力，但后来吴遭到来自中<span class='wp_keywordlink'><a href="https://www.bannedbook.org/forum24/" title="国学传统文化禁书" target="_blank">国学</a></span>生邹宏基 ( Hongji Zou ) 强行拖出现场，事件引发广泛关注。</p> <p>吴可薇在活动中高喊 : “你剥夺了香港人民最基本的自由，摧毁了他们的民主 ! 现在，你企图在我的国家台湾做同样的事 !” 这段话让全场陷入短暂的混乱。据《华盛顿自由灯塔》报导，吴的抗议仅持续了 20 秒，但随即被邹宏基，一名来自哈<span class='wp_keywordlink'><a href="https://www.qi-gong.me/buddhism/" title="佛教" target="_blank">佛教</a></span>育学院的中国留学生，暴力拉离现场。邹当时是<a href="https://www.bannedbook.org/bnews/tag/%e4%b8%ad%e5%9b%bd%e5%ad%a6%e7%94%9f/" class="st_tag internal_tag" rel="tag" title="标签 中国学生 下的日志">中国学生</a>学者联谊会 ( CSSA ) 的成员，该组织与中国共产党有密切联系。</p> <p>根据影片显示，邹宏基在谢锋演讲期间，强行将吴脱出会场，甚至扭曲吴的手臂。该事件后来被哈佛校警纪录为“袭击”，并引起了众多美国政界人士的关注。</p> <p>根据众议院中国共产党问题特别委员会的调查结果，哈佛大学最终对这一事件的处理方式引发了广泛的争议。尽管吴可薇的抗议仅持续短暂时间，且她随即道歉，但哈佛大学却在 5 月对吴施以纪律处分，将她至于 “纪律观察” 状态。相反地，对于邹宏基的暴力行为，校方仅要求他提供道歉信，并未对其采取进一步惩处。</p>  <p>哈佛教育学院的学术院长马丁 · 韦斯特 ( Martin West ) 再给邹的信中指出，尽管邹的行为违反了哈佛的 “暴力行为政策” ，但由于他在事件后遭遇网络骚扰，且其个人信息被公开，校方决定部对他实施任何惩罚。</p> <figure><figcaption>哈佛大学认为邹宏基已经在网络上得到了惩罚，就不在对他加以除罚。 图片来源: X账号 @aaronsibarium</figcaption></figure> <p>这一处理结果引发了众多批评，众议院中国共产党问题特别委员会主席约翰 · 穆勒纳 ( John Moolenaar ) 指出 : “哈佛惩罚了那些勇敢发声反对中共人权侵犯的学生，却对袭击者网开一面，甚至向他道歉。”</p> <p>报导指出，这起事件与中国留学生在美国校园内的行为模式高度一致。美国国务院表示，中国学生学者联谊会 ( CSSA ) 在美国各大校园内监视中国学生的行为，并报告给北京当局。类似的恐吓与骚扰事件并不罕见，例如，今年 4 月一名中国籍学生因在波士顿柏克利音乐学院威胁另一名学生而被判刑。</p> <p>这起事件让美国国会更加关注中国政府对美国校园内学生的影响，并引发了对哈佛大学校方在处理此类事件上的双重标准的质疑。《华盛顿自由灯塔》报导认为，哈佛在应对这类抗议与学生行为的不一致处理，可能反映了校方对不同政治立场的抗议采取了选择性执法的态度。</p>  <p>这起事件加剧了哈佛大学内部与外部对学校处理抗议行为的批评，特别是在针对中国政府的抗议与其他形式的抗议之间所显示的惩处差异。当校方面对国际政治与人权问题，哈佛如何保持其自由言论与公平执法的声誉，仍是一个值得深思的问题。</p> <p></p> <p>来自台湾的学生吴可薇 ( Cosette Wu )
