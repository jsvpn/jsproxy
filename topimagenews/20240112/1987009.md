id="conimg">中国半官方智库“南海战略态势感知计划平台”(SCSPI)发布美军“卡尔文森号(USS Carl Vinson CVN-70)”航舰航迹图。&nbsp;&nbsp;&nbsp;图：翻摄“X”@SCS_PI</p> <p>美军航母卡尔．文森号战斗群于本月月初与菲律宾的船舰在南海进行共同巡航的活动。在活动结束后，卡尔．文森号上的美国海军军官一同拜访了马尼拉。外界普遍猜测卡尔．文森号在结束了对马尼拉的访问后，会回到南海继续与解放军的海军进行对峙。然而根据中共半官方智库公布的航行轨迹判断，卡尔．文森号经由巴士海峡离开南海并驶向台湾东部海域，疑似为明日 ( 13 日 ) 举办的总统及立委选举做准备。</p> <p>在本月 3 日、 4 日两日，美国与菲律宾联合进行代号为“海上合作行动”( MCA ) 的联合巡航活动。巡航结束后，包含卡尔．文森号的美国船舰都靠岸马尼拉港并进行对菲律宾的访问。</p> <p>近日有情报传出，卡尔．文森号在离开马尼拉港后，并没有如同外界先前的预测，回到南海海域继续与解放军进行对峙，反而是向北行驶。根据中共半官方智库“南海战略态势感知计划平台”( SCSPI ) 公布对美军舰载运输机 C-2A 的飞行轨迹判断，卡尔．文森号应该是通过巴士海峡后，往台湾东海的方向驶去。但并没有证据表明卡尔．文森号目前正在台湾东部海域停留。</p>  <p>这段航行轨迹引发了台海两岸网民的热议，对岸网民猜测卡尔．文森号是要为绿营的选情站台，让台独的势力能在这次大选中再次取得胜利。然而<span class='wp_keywordlink_affiliate'><a href="https://www.bannedbook.org/" title="中国" target="_blank">中国</a></span>军事专栏作家《张学峰看空天》认为，美军航母在这个敏感时机点出现在这个敏感地点，另一部分的原因是想吓阻中国解放军，避免中国在绿营大选胜利后，立刻组织针对台湾的攻击。</p> <p>同时，也有台湾网友猜测美军航母是不是跟前几天发生的国家警报事件有关。但国防部已于今日下午发布声明表示，美军的航母行动路线与国家警报事件无关，请民众不要做过多的猜测。</p> <p>另外根据《香港01》的报导，美国预计在台湾大选后派遣前副国务卿詹姆士．斯坦伯格 ( James Steinberg ) 与共和党的前国家安全顾问史蒂芬．哈德利 ( Stephen Hadley ) 率团拜访台北。前立法委员郭正亮也表示，这代表着美国的立场是支持台海稳定维持现状。</p>  <p>美国卡尔．文森号航母在选前通过巴士海峡前往台湾东部海域，真实目的引发网友猜测。图为卡尔．文森号示意图。&nbsp;&nbsp;&nbsp;图：翻摄“X”@AIYeyENGDdJkjTr</p> <!--<div id="taboola-mid-1"></div>--><ul
