href="https://www.bannedbook.org/bnews/tag/%e7%be%8e%e5%9b%bd/" class="st_tag internal_tag" rel="tag" title="标签 美国 下的日志">美国</a>入境要求，从1月26日开始，乘坐国际航班到达美国的旅客，需要出示在出发前72小时内的COVID-19（中共<a href="https://www.bannedbook.org/bnews/tag/%e7%97%85%e6%af%92/" class="st_tag internal_tag" rel="tag" title="标签 病毒 下的日志">病毒</a>）核酸检测阴性测试。</p> <p>对于<a href="https://www.bannedbook.org/bnews/tag/%e5%a2%a8%e8%a5%bf%e5%93%a5/" class="st_tag internal_tag" rel="tag" title="标签 墨西哥 下的日志">墨西哥</a>和加勒比海地区的旅游业来讲，美国<a href="https://www.bannedbook.org/bnews/tag/%e5%ae%a2%e4%ba%ba/" class="st_tag internal_tag" rel="tag" title="标签 客人 下的日志">客人</a>是他们生意的一个主要来源。因此其中的一些高档酒店和<a href="https://www.bannedbook.org/bnews/tag/%E5%BA%A6%E5%81%87%E6%9D%91/" class="st_tag internal_tag" rel="tag" title="标签 度假村 下的日志">度假村</a>集团正在启动现场COVID-19测试，以提高顾客的安全性和安心感。 以下是这些着名<a href="https://www.bannedbook.org/bnews/tag/%E5%93%81%E7%89%8C/" class="st_tag internal_tag" rel="tag" title="标签 品牌 下的日志">品牌</a>在今年冬天推出的针对客人的现场病毒测试服务。</p> <ul> <li> <h2>AIC酒店集团</h2> </li> </ul> <p></p> <p>AIC酒店集团在美国和整个墨西哥和加勒比海地区拥有豪华度假酒店物业，例如Eden Roc迈阿密海滩、Nobu酒店迈阿密海滩和Nobu酒店芝加哥、以及包罗万象的Hard Rock酒店等。</p> <p>其将为所有在2021年2月28日之前预订三晚或以上住宿的美国居民提供免费的COVID-19测试和免费医疗援助。 符合条件的客人每间客房将获得两次免费的抗原检测，并可以以$ 50的价格购买其他检测。</p> <ul> <li> <h2>普拉亚（Playa）酒店及度假村</h2> </li> </ul> <p></p> <p>Playa度假品牌在墨西哥、和加勒比海地区拥有多家凯悦、希尔顿酒店及全包度假村等。其将为两地17个度假村中的任何一个返回美国的入住客人提供免费的COVID-19抗原检测。</p> <p>对于在墨西哥和多米尼加共和国至少住三晚以及在牙买加至少住五晚的客人，则可以享受免费现场抗原测试和PCR折扣测试，这些测试将于2021年1月22日至4月30日进行。 如果结果为阳性并需要进行隔离，则客人将获得50％的住宿折扣。</p> <ul> <li> <h2>Palace &amp; Le Blanc Spa度假村</h2> </li> </ul> <p></p>  <p>Palace&amp;Le Blanc Spa度假胜地最近扩展了Purely计划，为返回美国的客人提供免费的现场COVID-19抗原检测。旅行者可以在办理入住手续时安排现场测试，并期望在30分钟内得到结果。 客人还可以通过第三方安排现场PCR检测。</p> <ul> <li> <h2>Sandals&amp;Beaches度假村</h2> </li> </ul> <p></p> <p>Sandals&amp;Beaches 度假村将在2021年3月31日之前，对客人开放“由经认可和执业的医疗专业人员进行的”提供的现场 COVID-19测试，由酒店承担费用。</p> <ul> <li> <h2>LaColección度假村</h2> </li> </ul> <p></p> <p>La Coleccion 度假村将免费提供COVID-19抗原检测和PCR检测。 该品牌的“充满信心的旅行保护”计划还包括从登记到退房的COVID-19检测，以及长达72小时的远程医疗帮助。 检出COVID-19阳性的客人在隔离时将获得优惠折扣价。</p> <ul> <li> <h2>Palladium 酒店集团</h2> </li> </ul> <p></p> <p>Palladium酒店集团将在客人出发前向其客人提供COVID-19 PCR检测服务。 客人可以在办理入住手续时安排预约，并在72小时内会得到结果。 更重要的是，在美洲Palladium 酒店的住客可以享受免费医疗保险，包括医疗接送，长期住宿优惠等。</p> <ul> <li> <h2>AM度假村集团（AMResorts）</h2> </li> </ul> <p></p> <p>豪华度假品牌AM集团将为客人提供免费的现场COVID-19抗原检测以及隔离区长期住宿服务。 该计划将于1月26日启动，有效期至2021年3月31日。如果客人的冠状病毒检测呈阳性，则AMResorts将支付部分的14天检疫费用。</p>  <ul> <li> <h2>卓越度假村（Excellence Resorts）</h2> </li> </ul> <p></p> <p>多米尼加共和国，牙买加和墨西哥的Excellence Resorts酒店正在与当地实验室合作，如果客人返回原住国需要核酸检测结果时，酒店可以为客人提供现场的免费COVID-19测试。</p> <h2> </h2> <ul> <li> <h2>椰子湾海滩度假酒店（Coconut Bay Beach Resort）</h2> </li> </ul> <p></p> <p>圣卢西亚的椰子湾海滩温泉度假酒店和椰子湾的姊妹度假胜地Serenity均为入住五晚或以上的客人在出发前提供免费的COVID-19抗原检测。 同时，入住少于五晚的客人可以每人75美元的价格进行测试。</p> <ul> <li> <h2>Karisma酒店及度假村</h2> </li> </ul> <p></p> <p>Karisma Hotels＆Resorts正在其位于里维埃拉玛雅、墨西哥、多米尼加共和国和牙买加的场所进行免费的现场COVID-19抗原检测。 如果结果是阳性，该品牌还将为旅客提供长达14晚的优惠隔离价格。</p> <ul> <li> <h2>蓝钻石度假村（Blue Diamond Resorts）</h2> </li> </ul> <p></p> <p>从2021年1月26日开始，该集团下属的Royalton豪华度假村等将为度假村的游客提供免费的现场COVID-19病毒抗原测试。这些游客可在2021年3月31日之前获得免费的现场病毒抗原测试。</p> <ul> <li> <h2>好莱坞星球海滩度假村 （Planet Hollywood Beach Resorts）</h2> </li> </ul> <p></p>  <p>从2021年1月26日开始，好莱坞星球海滩度假村将提供现场COVID-19 PCR和抗原测试。哥斯达黎加度假村的客人可以以150美元的价格安排一次PCR测试，而坎昆度假村的游客则计划在2021年1月29日开始为客人安排免费病毒抗原测试，并在当天取结果。</p> <ul> <li> <h2>巴伊亚普林西比（Bahia Principe）酒店及度假村</h2> </li> </ul> <p></p> <p>Bahia Principe酒店及度假村向来美国旅行的客人免费提供病毒抗原检测。 免费的抗原测试将于2021年1月23日至2021年3月31日，或直到有关此新要求的法规变更之前可用，并且将在72小时内以书面和/或电子格式接收结果。</p> <ul> <li> <h2>美利亚国际酒店（MeliáHotels International）</h2> </li> </ul> <p></p> <p>从2021年1月20日开始，在墨西哥和多米尼加共和国的10家美利亚国际酒店中，如有客人返回原住国需要出示核酸结果，如果他们在该酒店入住至少三晚，则可以得到免费现场COVID-19抗原检测。 如果需要，度假村还将与当地的测试站点协调PCR测试。</p> <ul> <li> <h2>伊贝罗斯塔集团 (Iberostar Group)</h2> </li> </ul> <p></p> <p>从1月26日至2021年3月31日，Iberostar集团将在墨西哥、牙买加、多米尼加共和国和巴西的下属酒店中，为客人提供现场免费的COVID-19抗原测试。</p> <p>本文章或节目经希望之声编辑制作，转载请注明希望之声并包含原文标题及链接。</p> <ul class='op-related-articles' title='相关阅读'> <li><a
