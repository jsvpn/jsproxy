<h3>了解入境要求</h3> </li> </ul> <p><a href="https://www.bannedbook.org/bnews/tag/%E4%B9%98%E9%A3%9E%E6%9C%BA/" class="st_tag internal_tag" rel="tag" title="标签 乘飞机 <p>加拿大是目前全球为数不多的、边境防疫措施很严格的国家之一。乘坐航班进入加拿大的人，在航班上要全程佩戴口罩，且必须在抵达加前72小时内使用ArriveCAN应用程序或网站分享他们的旅行详情；即使是完全接种了疫苗，也可能再被要求做入境时的病毒检测。</p>  <p>出发前，请登陆相关官方网站，以了解最新入境须知。</p> <ul> <li> <h3>提前计划</h3> </li> </ul> <p>提前了解有关停车场和车库可用性的信息。至少在国内航班前两小时和国际航班前三小时到达机场。</p> <p>提前做好功课，如果遇到航班延误或取消，你清楚与哪些航空公司联系，以不耽搁行程。</p>  <ul> <li> <h3>使用社交媒体、应用程序</h3> </li> </ul> <p>建议安装航空公司的应用程序，这样你可以在其社交媒体上关注航空公司的航班动态。如果出现问题，您可以自己在App重新预订航班。</p> <ul> <li> <h3>尽量避开一些机场</h3> </li> </ul> <p>某些机场的延误可能比其他机场多。例如，据 FlightAware数据，今年夏季，加拿大多伦多皮尔逊国际机场、美国奥兰多国际机场排在全球航班延误比例最高的前十名机场之列。</p>
