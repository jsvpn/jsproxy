target="_blank">中国</a></span>在西方各大学设置“孔子学院”，表面上是为了推广<span class='wp_keywordlink'><a href="https://www.bannedbook.org/forum24/" title="国学传统文化" target="_blank">中国文化</a></span>【小编推荐:<a href='https://www.bannedbook.org/bnews/comments/20220808/1768773.html' target='_blank'>探寻复兴中华之路，必看章天亮博士《中华文明史》</a>】和语言，实际上是作为宣传中共体制、监控中国留学生的据点。而根据美国草根教育组织“家长捍卫教育”（PDE）7月26日发布的调查报告，中国的这类洗脑机构如今已渗透美国的K-12（幼儿园至高中），引发家长和国会议员忧心忡忡，认为必须尽速采取驱逐措施。</p> <p>《自由时报》综合“大众<span class='wp_keywordlink_affiliate'><a href="https://www.bannedbook.org/" title="新闻">新闻</a></span>”等媒体报导，根据PDE发布的“小红教室”（Little Red Classrooms）调查报告，中共与34州和哥伦比亚特区的143个学区签署设立孔子学院（Confucius Institutes）和孔子教室（Confucius Classrooms）合约，这143个学区中的20个位于美国军事基地附近，其中还包括美国3所顶尖的<span class='wp_keywordlink'><a href="https://www.bannedbook.org/forum11/topic309.html" title="禁片：“科学”的棍子" target="_blank">科学</a></span>高中。</p> <p>报告指出，自2009年起的14年间，中国政府向这143个学区资助了约1800万美元资金，在这个过程中，中国政府还干预了在美国中小学教授中文、文化、历史等课程的中国籍教师的遴选。</p>  <p>PDE在报告中举例说，2012至2013学年，2名中国公民在德拉瓦州Appoquinimink学区教授华语，根据学区官网，这2名教师在中国经历了严格的遴选程序，包括接受中国政府官员的面试。</p> <p>PDE指出：“美国的低龄学生现在成为中共宣传理念的目标，得知中国政府一直干预低年级学生的教学课程，这令美国很多家庭感到震惊和担忧。”</p>  <p>PDE高级顾问艾克斯纳（Michele P. Exner）在致联邦众议院外交委员会主席麦考尔（Mike McCaul）的信函中指出：“中国政府对美国教室、学生和课程有如此多的接触权，这个事实应该让家庭感到担忧。34州和哥伦比亚特区的学校与中国政府或其附属实体进行了每份合约金额，从几千美元到超过100万美元的金融往来。”</p> <p>艾克斯纳写道：“自2009年以来，中国政府利用孔子学院和孔子教室在全球学校推进其议程，中共官员公开表示，他们如何利用这些以文化和语言沉浸计画为幌子，所建立的教育伙伴关系，扩大中国的海外宣传活动。”</p>  <p>除了麦考尔，PDE还致函多名参众两院的重量级议员，以及事涉的34州州长，要求他们关注和调查此事。而共和党众议院班克斯（Jim Banks）在PDE发布调查报告后，就立即致函教育部长卡多纳（Miguel A. Cardona），信函称“中共的钱不该流入美国的教育机构”，要求教育部展开全面调查。</p> <!--<div id="taboola-mid-1"></div>--><p class="texttj"> 🔥<a href="https://www.bannedbook.org/bnews/ssgc/20230219/1850782.html" target="_blank">法国犹太老板：神告诉我们，只有一位中国人能救人类</a><br/> 🔥<a href="https://www.bannedbook.org/bnews/comments/20220220/1694796.html" target="_blank">华人必看：中华文化的飓风 幸福感无法描述</a><br/> 🔥<a href="https://github.com/bannedbook/fanqiang/wiki/V2ray%E6%9C%BA%E5%9C%BA" target="_blank">解锁ChatGPT|全平台高速翻墙:高清视频秒开,超低延迟</a><br/> 🔥<a href="https://www.bannedbook.org/bnews/comments/20220808/1768773.html" target="_blank">探寻中华复兴之路，必看章天亮博士《中华文明史》</a><br/> <a href="https://github.com/bannedbook/fanqiang/wiki/%E7%A6%81%E9%97%BB%E7%BD%91%E5%AE%89%E5%8D%93%E7%BF%BB%E5%A2%99%E6%96%B0%E9%97%BBAPP" target="_blank">免费PC翻墙、安卓VPN翻墙APP</a><br/> </p><p>2005年中国首家孔子学院在马里兰大学设立，此后持续增加，2017年增至118所。但川普政府执政期间，因美中矛盾加剧，要求各大学关闭，截至2022年底，美国境内孔子学院已降至7所。今年美国参、众两院提出禁止向设立孔子学院的大学提供资金的法案。#</p><a name='sharetosocial'></a> <div style="margin-bottom:5px;padding-bottom:5px;clear:both"> <div id="archive-pix-1" class="banner-ads"> <!-- AuctionX Display platform tag START --> <div id="27602x728x90x621x_ADSLOT1" clicktrack="%%CLICK_URL_ESC%%"></div>  <!-- AuctionX Display platform tag END --> </div> <div id="archive-pix-2" class="banner-ads"> <!-- AuctionX Display platform tag START --> <div id="27556x300x250x621x_ADSLOT1" clicktrack="%%CLICK_URL_ESC%%" style="margin:0 auto;text-align:center"></div>  <!-- AuctionX Display platform tag END --> </div> </div>  <div id="archive-pix-1" class="banner-ads"> <!-- AuctionX Display platform tag START --> <div id="27603x728x90x621x_ADSLOT1" clicktrack="%%CLICK_URL_ESC%%"></div>  <!-- AuctionX Display platform tag END --> </div> </div><!--END ENTRY-->
