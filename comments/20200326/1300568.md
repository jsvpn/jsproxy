<p>根据记者会上公布的信息，以下是2万亿美元的经济援助计划包括的<wbr></wbr>一些项目：</p> <ul> <li>对于大多数年收入不超过7万5千元的个人，直接付款$ 1,200；</li> <li>对于年收入不超过15万元的夫妻，直接付款为$ 2,400。每个子女会额外增加$ 500。</li> <li>对于收入超过7万5千元的个人，金额减少。   </li> <li>对于收入超过9万9千元的个人，不能获得此项救济金。</li> </ul> <ul> <li>扩大的失业救济金2亿5千万元，使最高救济金每周增加600元，<wbr></wbr>并为下岗工人提供四个月的全薪。</li> <li><wbr></wbr>独立承包商和个体经营者也符合资格。</li> </ul> <ul> <li>小企业贷款3670亿元</li> <li>1500亿元给州和地方政府抗击疫情</li> <li>1300亿元用于医院</li> <li>160亿元买个人防护设备</li> </ul> <p>为较大的行业提供5000亿元的贷款，<wbr></wbr>其中包括：</p>  <ul> <li>为客运航空公司提供的250亿美元；</li> <li>运输公司40亿元；</li> <li>30亿元用于航空承包商，</li> <li>170亿元用于“<wbr></wbr>对维护国家安全至关重要的业务”。</li> </ul> <p>建立监督委员会和监察长，监督对大公司的贷款，<wbr></wbr>不允许公司用于买回股票（stock buyback）。</p> <p>此外， </p> <ul> <li>230亿元充实战略储备 </li> <li>4亿元用于选举安全津贴 </li> <li>2500万元，用于约翰·肯尼迪表演艺术中心 </li> <li>措施禁止川普总统及其家人拥有的公司获得联邦救济</li> </ul> <p>美国财政部长姆努钦表示，三周内，给美国民众的救助金会直接付款会到位；到下周末，所有银行可以开始处理小商业救助贷款。</p>  <p>有关这个经济援助法案的协议文本预计要到周三晚些时候才会公开。</p>
