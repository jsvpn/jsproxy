href="https://www.bannedbook.org/bnews/tag/%E5%B7%9D%E6%99%AE%E6%80%BB%E7%BB%9F/" class="st_tag internal_tag" rel="tag" title="标签 川普总统 下的日志">川普总统</a>承认耶路撒冷为以色列首都、下令将<a <p>川普总统第一期总统任期期间中东政策大事记：</p> <ul> <li>2017年1月27日，川普总统签署立即生效的行政令，以阻止极端伊斯兰恐怖分子入境美国为由，禁止类似于伊拉克、叙利亚等主要是穆斯林国家的人前往美国</li> <li>2017年4月7日，叙利亚内战期间</li> </ul> <p>川普总统下令美国海军从位于地中海的伯克级驱逐舰发射了59枚战斧巡航导弹， 打击阿萨德政府控制的沙伊拉特空军基地（Shayrat Air Base）以惩罚阿萨德政府对叙利亚人使用化学武器的行为</p> <ul> <li>2017年5月20日</li> </ul> <p>川普总统的就职后的首次外访的目的地是沙特阿拉伯，并在利雅得（Riyadh）参加了一个阿拉伯峰会，敦促穆斯林世界团结起来打击恐怖主义</p> <ul> <li>2017年12月6日</li> </ul> <p>川普总统承认耶路撒冷是以色列首都，</p> <ul> <li>2018年4月13日</li> </ul> <p>川普总统由于阿萨德政府反复对该国平民使用化学武器对叙利亚进行了更多次数的空袭</p>  <ul> <li>2018年5月8日</li> </ul> <p>川普总统谴责<a href="https://www.bannedbook.org/bnews/tag/%e4%bc%8a%e6%9c%97/" class="st_tag internal_tag" rel="tag" title="标签 伊朗 下的日志">伊朗</a>违反了伊朗核协议，宣布退出伊朗核协议，并重启对伊朗的制裁</p> <p>川普总统表示，伊朗核协议 “烂透了”，令包括他自己在内的所有美国公民蒙羞，因为该协议对于伊朗的弹道导弹项目、2025年以后的核活动以及在伊朗在中东地区的破坏行动没有任何制约作用，而这其实是川普总统在兑现他2016年参选时做出的一个承诺，他在2016年参选时就承诺，若他当选，美国会退出伊朗核协议。</p> <ul> <li>2018年5月14日</li> </ul> <p>美国将驻以色列大使馆从特拉维夫迁往耶路撒冷，</p> <p>这是美国数位前任总统在参选时做出的、但是就职后都没有兑现的承诺，因此被视为是他们的嘴皮子承诺，而川普总统在2016年参选是做出了此承诺，随后也于5月14日兑现了此承诺。</p> <ul> <li>2018年12月19日</li> </ul> <p>川普总统宣布从叙利亚撤出2000名美军</p> <ul> <li>2019年3月</li> </ul> <p>川普总统在白宫签署公告，正式承认以色列对戈兰高地的主权</p>  <ul> <li>2019年10月26日</li> </ul> <p>美军在叙利亚北部的空袭击毙了伊斯兰国的首领巴格达迪（Abu Bakr al-Baghdadi），巴格达迪之死意味着长达5年的伊斯兰国的灭亡， 结束了叙利亚、伊拉克北部地区的战争灾难，从某种程度上也制止了伊斯兰国恐怖分子在欧洲发起的恐袭。在此之前， 伊斯兰国的追随者曾经在欧洲的巴黎、布鲁塞尔、伦敦和尼斯发起数起恐袭，搞得欧洲<span class='wp_keywordlink_affiliate'><a href="https://www.bannedbook.org/" title="大陆" target="_blank">大陆</a></span>人心惶惶。 </p> <ul> <li>2020年1月3日</li> </ul> <p>美军用无人机击毙了伊朗革命卫队的将领苏莱曼尼</p> <p>此举被视为是川普总统外交政策中最大胆之举，因为美军此次空袭一次性地击毙了伊朗军方的一个实权人物， 虽然这被视为是一场非常危险的赌博，但是很多人认为这是值得的。 英国布莱尔研究所（the Tony Blair Institute）伊朗问题分析师阿拉比（Kasra Aarabi）说：“当美国击毙了苏莱曼尼后，伊朗政坛确实一度出现了真空，因为苏莱曼尼是伊朗破坏中东地区稳定的幕后策划者， 他还应该对中东地区发生的很多流血事件负责。”</p> <ul> <li>2020年1月28日</li> </ul> <p>川普总统在未与巴勒斯坦当局讨论的情况下，与以色列总理内塔尼亚胡公布了其政府就中东问题的和平方案</p> <ul> <li>2020年9月15日</li> </ul> <p>川普总统在白宫目击以色列、巴林和阿联酋（UAE）三国领袖签署了《亚伯拉罕协议》（Abraham Accords，又称《以阿和平协议》），这一历史性协议令阿联酋、以色列和巴林三国的关系正常化，三国互建使馆，并扩展了三国之间的贸易，而这三个阿拉伯国家，甚至更多的阿拉伯国家都都希望能够据此协议促成在中东地区建立一个针对伊朗的、类似于北约的盟友关系，因此川普总统曾经表示， 还会可有更多的阿拉伯国家加盟此协议。</p> <p>川普总统称：此协议的签署“是中东地区和平的新曙光”。</p>  <p>国际社会对于这个《亚伯拉罕协议》的评价也很高，因为此协议绕过了难缠的巴勒斯坦， 让以色列和阿拉伯国家直接签署了协议，《纽约观察报》编辑库尔森曾经表示， 在过去长达六十年的时间中，美国曾经多次向中东派去很多优秀的外交官，希望能够实现以色列和阿拉伯国家的和解，都无果， 然而川普政府却做到了。</p> <p>奥巴马政府时期的美国前国务卿 凯瑞（John Kerry）就曾经于2016年表示，以色列不可能与阿拉伯国家签署核协议，而奥巴马政府当年推动的中东谈判也停滞不前。 当时凯瑞曾说：“没有巴勒斯坦的和平， 就无法得到阿拉伯世界的和平，这是一个非常严酷的现实。”但是川普政府做到了。</p> <p>当地时间2020年6月14日， 以色列屯垦部长霍托维利（ Tzipi Hotovely ）将在戈兰高地一块以色列占领区的定居点，作为送给美国川普总统74岁的生日礼物， 命名为川普高地（Trump Height），以此来感激川普总统承认耶路撒冷为以色列首都，将美国驻以色列大使馆从特拉维夫迁往耶路撒冷， 以及承认以色列对戈兰高地的主权；不仅仅如此， 川普总统还于2018年5月8日退出了伊朗核协议并重新启动了对伊朗对全面制裁， 这个立场也令直接受到伊朗威胁的以色列政府和民众非常感激。 </p> <p>文章认为， 川普总统在其第一期总统任期期间， 颠覆了美国政府过去几十年的中东政策，将海湾国家视为朋友，同时在以色列与巴勒斯坦之间长达数百年的冲突中，公开支持以色列，此举不仅为他赢得了美国共和党中的右翼选民的支持和他自己的福音派选民的支持，也赢得了以色列民众的支持。  川普高地当地的一位官员，前以色列国防军官员罗卡奇（Haim Rokach）认为，川普总统并不仅仅是一位兑现了其竞选承诺的总统；他认为川普总统在第一任期期间为以色列做的一切在兑现《圣经》中的一个承诺，令以色列人距在耶路撒冷重建第二座圣殿的时间不远了。</p> <p>本文章或节目经希望之声编辑制作，转载请注明希望之声并包含原文标题及链接。</p> <ul class='op-related-articles' title='相关阅读'> <li><a
